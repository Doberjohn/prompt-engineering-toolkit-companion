UI/UX URL Evaluator

Audit a live URL against Nielsen's 10 Usability Heuristics, WCAG 2.2 AA, and a 20-dimension scoring system covering both UI (objective) and UX (heuristic inference). Produces severity findings with recommended fixes.

How URL evaluation works The evaluator does not fetch the URL itself. It works from the URL and any extra context you provide (target audience, user flows, known constraints). For the deepest audit, add a short description of what the page is for and who uses it.
